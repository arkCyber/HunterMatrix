常见问题 FAQ

HunterMatrix基于TauriFrameworkDevelopment，支持窗口s、macOS和LinuxOperationSystem。 具体支持Version：窗口s 10/11、macOS 10.15+、Ubuntu 18.04+等主流Linux发行版。

病毒Library会自动更新，您也可以Manual更新：
1. 在仪表板点击"更新病毒库"按钮
2. 或在设置页面启用"自动更新病毒库"
3. System会从HunterMatrix官方服务器下载最新病毒特性Library

优化扫描速度的Method：
1. 在扫描设置中增加扫描Thread数
2. 排除不需要扫描的大File或Directory
3. 使用快速扫描模式而非完整扫描
4. 确保System有足够的可用Memory

AI功能基于本地Ollama服务Run，不需要将数据Send到云端：
1. 首次使用需要下载AIModel（需要联网）
2. Model下载完成后可离线使用
3. 威胁情报更新需要联网获取最新数据
4. 所有Analysis都在本地进行，保护隐私安全

配置邮件通知步骤：
1. 进入设置页面的"邮件设置"选项卡
2. 选择邮件服务商（Gmail、Outlook等）
3. 输入发件人邮箱和应用专用Password
4. 添加收件人邮箱地址
5. 点击"发送测试邮件"验证配置

处理病毒的建议步骤：
1. 查看扫描结果中的威胁详情
2. 选择隔离或删除恶意文件
3. 检查是否有其他相关威胁
4. 更新病毒库并重新扫描
5. 咨询AI助手获取专业建议
