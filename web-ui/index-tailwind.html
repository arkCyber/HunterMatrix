常见问题 FAQ

HunterMatrix基于TauriFrameworkDevelopment，支持Windows、macOS和Linux操作系统。 具体支持Version：Windows 10/11、macOS 10.15+、Ubuntu 18.04+等主流Linux发行版。

病毒库会自动更新，您也可以手动更新：
1. 在仪Table板点击"更新病毒库"按钮
2. 或在设置页面启用"自动更新病毒库"
3. 系统会从HunterMatrix官方服务器下载最新病毒Feature库

Optimization扫描速度的Method：
1. 在扫描设置中增加扫描Thread数
2. 排除不需要扫描的大文件或Directory
3. 使用快速扫描模式而非完整扫描
4. 确保系统Has足够的可用Memory

AI功能基于本地Ollama服务Run，不需要将数据发送到云端：
1. 首次使用需要下载AIModel（需要联网）
2. Model下载完成后可离线使用
3. 威胁情报更新需要联网获取最新数据
4. 所Has分析都在本地进行，保护隐私安全

配置邮件通知步骤：
1. 进入设置页面的"邮件设置"Options卡
2. 选择邮件服务商（Gmail、Outlook等）
3. Input发件人邮箱和Application专用Password
4. 添加收件人邮箱地址
5. 点击"发送测试邮件"验证配置

处理病毒的建议步骤：
1. 查看扫描结果中的威胁详情
2. 选择隔离或删除恶意文件
3. 检查是否Has其他相关威胁
4. 更新病毒库并重新扫描
5. 咨询AI助手获取专业建议
